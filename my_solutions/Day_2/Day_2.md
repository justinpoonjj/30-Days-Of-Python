## Variables: 

Store data in computer memory. 

Mnenmonic variable (recommended): A variable name that can be easily remembered and associated
    - A variable refers to a memory address which data is stored
    - variable can have short name but a more descriptive name is recommended

Variable name rules: 
    - Start with a letter or underscore char
    - Cannot start with a number
    - Can only contain alpha-numeric characters and underscores
    - Names are case-sensitive

Standard naming style: 
    - snake_case variable naming convention
        - Use underscore character after each word for a variable containing more than one word
        eg. first_name, last_name

Variable declaration: assign a data type to a variable using "="
